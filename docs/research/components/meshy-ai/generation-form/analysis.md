# Meshy AI Generation Form Notes

Source reviewed: https://www.meshy.ai

## Prompt Textarea
- Large primary input area with generous vertical space.
- Dark surface contrast: panel and field are visually separated.
- Placeholder encourages concise but descriptive prompt text.

## Generate Buttons
- Main CTA uses high-contrast gradient/accent color.
- Rounded corners (not fully pill) and strong hover state.
- Disabled/loading state reduces interaction and changes label.

## Loading Indicator
- Compact inline loading element near CTA.
- Spinner + short status text.
- Kept visible while generation is in progress.

## Style Selectors
- Compact selects grouped below prompt.
- Clear labels and concise option lists.
- Consistent sizing and spacing with prompt + button stack.
